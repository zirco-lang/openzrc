# Stage 1 Zirco Implementation

This is the first stage of openzrc compilation. This is a compiler written purely in C that only implements a core subset of zirco.

It should support the following:

 - one-line comments
 - identifiers
 - integer literals (decimal only)
 - Boolean literals
 - Character Literals.
 - Arithmetic, comparison, logical, bitwise, assignment, and `as` operators
 - delimeters

# supported types
 - i\*
 - u\*
 -  Pointers.


# Supported Keywords
 - fn
 - if
 - do
 - while
 - let
 - return
 - break
